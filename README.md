# Magnifico Tools — Claude Code Plugin Marketplace

A curated collection of Claude Code plugins by [Magnifico4625](https://github.com/Magnifico4625).

## Install

```bash
/plugin marketplace add Magnifico4625/claude-plugins
```

## Available Plugins

| Plugin | Version | Description |
|--------|---------|-------------|
| [locus](https://github.com/Magnifico4625/locus) | 3.1.0 | Persistent project-aware memory for Claude Code and Codex CLI. Client-aware storage, structural map, decisions, conversation history. 863 tests, 12 MCP tools. |

## Install a Plugin

After adding the marketplace:

```bash
/plugin install locus@magnifico-tools
```
